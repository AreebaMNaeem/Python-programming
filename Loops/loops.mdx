## 🔶 What is a Loop and Why Do We Need It?

Imagine you have 100 students and you want to print the name of each student. Without a loop you'd write 100 print statements. That's insane.  
A loop lets you write the logic **once** and repeat it automatically until a specific condition is met.

### ➡️ Without a loop:
```python
print("Areeba")
print("Alishba")
print("Zara")
# ... 97 more lines
```

### ➡️ With a loop :
```python
students = ["Areeba", "Alishba", "Zara"]
for student in students:
    print(student)
```
Same result. One piece of logic. Works for 3 or 3 million names.

## 🔶 The Two Types of Loops in Python : 

### 1 .while loop → condition-driven
"keep going WHILE something is True" . Use when you don't know how many times to repeat.

**Example:**
```python
count = 1
while count <= 5:
    print("Number:", count)
    count += 1
```
### 2. for loop → iteration-driven
"do this FOR each item in a collection" . Use when the number of iterations is known.

**Example:**
```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print("I like", fruit)
```
## 🔶 What is an Iterable?
An iterable is anything Python can loop over — anything that can hand out its items one by one.

### Built-in iterables:

🔻 tuple

🔻 string

🔻 range

🔻 dict
