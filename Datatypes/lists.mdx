## 🔶 What is a List?

A list is a built-in data structure in Python used to store an ordered collection of items. Lists are dynamic, resizable, and capable of storing multiple data types in a single structure.

**Key characteristics:**

🔻 **Ordered** — elements maintain the order in which they were inserted

🔻 **Mutable** — values can be changed, added, or removed after creation

🔻 **Heterogeneous** — can store integers, strings, booleans, and even other lists

🔻 **Index-based** — elements are accessed using their position, starting from index 0

🔻 **Allows duplicates** — the same value can appear more than once

---

## 🔶 Creating a List :

### Using Square Brackets `[ ]`

The most common way to create a list:

```python
a = [1, 2, 3, 4, 5]               # List of integers
b = ['apple', 'banana', 'cherry']  # List of strings
c = [1, 'hello', 3.14, True]       # Mixed data types

print(a)  # [1, 2, 3, 4, 5]
print(c)  # [1, 'hello', 3.14, True]
```

### Using Repeated Elements

Use the `*` operator to fill a list with a repeated value:

```python
a = [2] * 5   # [2, 2, 2, 2, 2]
b = [0] * 7   # [0, 0, 0, 0, 0, 0, 0]
```
---

## 🔶 Accessing List Elements :

Python uses zero-based indexing. Negative indices count from the end.

```python
a = [10, 20, 30, 40, 50]

print(a[0])    # 10  — first element
print(a[-1])   # 50  — last element
print(a[1:4])  # [20, 30, 40]  — slicing
```

**Slicing tips:**
- `list[start:end]` — up to but not including `end`
- `list[::step]` — every step-th element
- `list[::-1]` — reverse the list

## 🔶 Adding Elements :

```python
# Adding Elements in a Books List :
books = []
books.append("Atomic Habits")                        # add to end → ['Atomic Habits']
books.insert(0, "Deep Work")                         # add to beginning → ['Deep Work', 'Atomic Habits']
books.extend(["The Alchemist", "Rich Dad Poor Dad"]) # add multiple
print(books)
# ['Deep Work', 'Atomic Habits', 'The Alchemist', 'Rich Dad Poor Dad']
```

### 🔶 Updating Elements :

Lists are mutable — assign a new value via index:

```python
# Online store — updating product price after discount :
prices = [1500, 2999, 450, 8000, 320]
prices[2] = 350    # price of item 3 updated after sale
print(prices)      # [1500, 2999, 350, 8000, 320]
```

### 🔶 Removing Elements :

```python
# Flight booking — passenger cancelled
passengers = ["Ali", "Sara", "Usman", "Zara", "Hamza"]
passengers.remove("Usman")   # specific person cancelled
last_added = passengers.pop()  # last person removed (waitlist logic)
print(passengers)  # ['Ali', 'Sara', 'Zara']
passengers.clear()           # removes ALL elements
```

## 🔶 Iterating Over a List :

### Basic `for` Loop

```python
fruits = ['apple', 'banana', 'cherry', 'mango']
for fruit in fruits:
    print(fruit)         # apple banana cherry mango
``` 

### Using `enumerate()` for Index + Value

```python
tasks = ['Wake up', 'Exercise', 'Study', 'Work']
for index, task in enumerate(tasks, start=1):
    print(f'{index}. {task}')

# Output:
# 1. Wake up
# 2. Exercise
# 3. Study
# 4. Work
```

### 🔶 Filtering with a Loop :

```python
numbers = [10, 15, 20, 25, 30, 35, 40]
even_numbers = []
for num in numbers:
    if num % 2 == 0:
        even_numbers.append(num)
print(even_numbers)  # [10, 20, 30, 40]
```

## 🔶 List Comprehension :

A concise, one-line way to create lists — more readable and often faster than traditional loops.

**Syntax:**

```python
new_list = [expression for item in iterable if condition]
```

### 🔻Squares 

```python
squares = [x**2 for x in range(1, 6)]
print(squares)  # [1, 4, 9, 16, 25]
```

### 🔻With Condition — Even Squares

```python
squares_of_evens = [x**2 for x in range(10) if x % 2 == 0]
print(squares_of_evens)  # [0, 4, 16, 36, 64]
```

### 🔻String Manipulation

```python
sentence = 'Python is fun and powerful'
word_lengths = [len(word) for word in sentence.split()]
print(word_lengths)  # [6, 2, 3, 3, 8]

uppercase_words = [word.upper() for word in sentence.split() if len(word) > 2]
print(uppercase_words)  # ['PYTHON', 'AND', 'POWERFUL']
```

### 🔻Flattening a Nested List

```python
nested = [[1, 2, 3], [4, 5], [6]]
flat = [item for sublist in nested for item in sublist]
print(flat)  # [1, 2, 3, 4, 5, 6]
```

### 🔻Multiple Conditions

```python
# Numbers from 1 to 50 divisible by BOTH 3 and 5
result = [x for x in range(1, 51) if x % 3 == 0 and x % 5 == 0]
print(result)  # [15, 30, 45]
```

## 🔶 Real-World Use Cases :

### 🔻Shopping List

```python
shopping_list = ['Milk', 'Eggs', 'Bread', 'Butter']
shopping_list.append('Cheese')
shopping_list.remove('Eggs')
print(shopping_list)  # ['Milk', 'Bread', 'Butter', 'Cheese']
```

### 🔻Bank Queue (FIFO)

```python
queue = ['Customer1', 'Customer2', 'Customer3']
queue.pop(0)  # first customer served
print(queue)  # ['Customer2', 'Customer3']
```

### 🔻Social Media Notifications

```python
notifications = ['New friend request', 'Message from Ali']
notifications.append('Someone liked your post')
print(notifications)
```

### 🔻Inventory Management

```python
inventory = ['Laptop', 'Mouse', 'Keyboard']
inventory.append('Monitor')  # restocked
inventory.remove('Mouse')    # sold out
print(inventory)  # ['Laptop', 'Keyboard', 'Monitor']
```
